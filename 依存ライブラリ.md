# バックエンド
##### Spring Boot
  - Webアプリケーションを素早く作るためのJavaフレームワーク
##### Spring WebSocket
  - サーバーとブラウザ間でリアルタイム通信を行う仕組み
##### Spring Messaging（STOMP Protocol）
  - WebSocket通信を宛名付きメッセージとして扱うための仕組み
##### Thymeleaf
  - HTMLを動的に生成するテンプレートエンジン

# フロントエンド
##### Thymeleaf
  - サーバー側でHTMLを組み立てるテンプレートエンジン
##### JavaScript（ES6）
  - ブラウザ上で動作や通信を制御するプログラミング言語
##### SockJS
  - WebSocketが使えない環境でも通信できる代替ライブラリ
##### Stomp.js
  - ブラウザでSTOMPプロトコルを扱うためのライブラリ
##### HTML / CSS
  - 画面構造と見た目を定義する技術
##### Bootstrap
  - 見た目を簡単に整えられるCSSフレームワーク
